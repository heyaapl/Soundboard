SOUNDBOARD ICON PLACEHOLDER

To add your custom minimap icon:

1. Create a 32x32 pixel PNG image
2. Name it "icon.png"
3. Place it in this folder (Soundboard/)
4. Delete this placeholder file
5. Type /reload in-game

Your custom icon will then appear on the minimap!

Current icon path: Interface\AddOns\Soundboard\icon
(WoW will automatically detect .png or .tga extension)
